## 🌍 Global Internship Finder

Global Internship Finder is a web application developed to help students explore internship opportunities across different countries. The project provides a simple, responsive, and user-friendly interface for searching internship information.

## 🚀 Features

- Search internship opportunities
- Clean and responsive user interface
- Easy navigation
- Modern design
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML
- CSS
- TypeScript
- Vite

## 📂 Project Structure

```
Global-Internship-Finder/
├── src/
├── guidelines/
├── index.html
├── package.json
├── vite.config.ts
└── README.md
```

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/vishnumithran-b/global-internship-finder.git
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```

Open the local development server URL shown in the terminal.
